Composer badge

How to add a badge of your package??

Copy address and append it with package name

Copy that image address. The main image address is:

http://composer.network/badge/[vendor]/[package]

Now append to this address name of package on packagist.org. For example,

http://composer.network/badge/wapmorgan/php-code-fixer

Change badge size, if it needs

You can use little version

http://composer.network/badge/wapmorgan/php-code-fixer/0

Or work with it's original size

http://composer.network/badge/wapmorgan/php-code-fixer/1

If you think that this is not enough, you can use badge with bigger size.

http://composer.network/badge/wapmorgan/php-code-fixer/2

Add extension

If you post badge on a site that requires it to have an extension, just add one of available extensions to badge address. Available extensions: .png, .svg, .gif. For example:

http://composer.network/badge/wapmorgan/php-code-fixer.png

or

http://composer.network/badge/wapmorgan/php-code-fixer/1.png
